# Detection d'Age par Intelligence Artificielle
Projet 10 - Deep Learning | Django + OpenCV + Caffe

## Installation
1. conda create -n age_detection python=3.11 -y
2. conda activate age_detection
3. pip install -r requirements.txt
4. python manage.py migrate
5. python manage.py runserver
6. Ouvrir http://127.0.0.1:8000

## Stack
- Python 3.11 + Django 5.2
- OpenCV DNN + Modeles Caffe (Face Net + Age Net)
- Analyse texture peau (Laplacien)
